# 数组 array
在Go中一个array的长度是不可变的，因此无法使用append操作。做为参数值在函数间传递时，也无法传递指针只能是拷贝一个新的数组。

声明：
```Go
//[3]int表示数组的元素为3个
var i [3]int
//或者推导
j := [3]int{}
```

赋值：
```Go
//可以在声明时赋值
var i [3]int = [3]int{1, 2, 3}
//或者推导
j := [3]int{1, 2, 3}
//也可以通过指定索引赋值
j[0]=11
```

判断长度（已赋值了的元素数量）：
```Go
j := [3]int{1, 2, 3}
len(j)
```

判断容量（可容纳的元素数量）：
```Go
j := [3]int{1, 2, 3}
len(j)
```
